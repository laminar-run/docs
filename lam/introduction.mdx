# lam Language

`lam` is a data transformation language that allows you to transform data from one format to another.




## Credits and License

Lam is a fork of the [vrl](https://github.com/vectordotdev/vrl) data transformation language.

Thanks to the VRL team for their work on the language. 

The `lam` language follows the same MPL 2.0 license as the VRL project.
